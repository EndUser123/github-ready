🛠️ Installation

Three deployment models:

  • SKILLS - Recommended for development (junction to ~/.claude/skills/)
  • HOOKS - For testing hook files (symlinks to ~/.claude/hooks/)
  • PLUGINS - For end user distribution (via /plugin command)
