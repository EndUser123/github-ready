System overview

The core flow is straightforward: detect package type, generate structure, add polish, validate outputs, and leave the repository in a publishable state.

Core capabilities

Type detection

Distinguishes Claude Code plugins, Claude skills, and Python libraries from the current repository shape.

Repository polish

Adds documentation, badges, GitHub Actions, and packaging conventions aimed at public presentation.

Media workflow

Uses NotebookLM and HTML docs pages so assets are usable on GitHub and GitHub Pages.

Brownfield conversion

Supports converting existing Python repositories into a plugin-oriented structure when appropriate.
